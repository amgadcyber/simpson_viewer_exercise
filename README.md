# Run
- you can pass the `-d` to specify the device to run on
### TO run variant one
```dart
   flutter run --flavor variant_one -t .\lib\main_variant_one.dart
```
### TO run variant two
```dart
   flutter run --flavor variant_two -t .\lib\main_variant_two.dart
```
